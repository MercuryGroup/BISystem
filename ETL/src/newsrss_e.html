getData/1

getData(Symbols::string()) -> ok

Sends a request to the server to retrieve news items (concurrently) for each provided stock/market symbol. Symbols = string() e.g. "yhoo,aapl,^ftse,3in.l,aak.st".

Example call setup: newsrss_e:getData("yhoo,aapl,^ftse,3in.l,aak.st").
